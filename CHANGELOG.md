# Changelog

## 0.2.0

- Renamed app to External Folder Refresh.
- Changed app id to external_folder_refresh.
- Added admin settings section.
- Added configurable allowed folder prefixes.
- Added default allowed prefix /Yandex.
- Added optional group restrictions.
- Added per-user and per-folder rate limiting.
- Added scan timeout.
- Added lock file to prevent parallel scan of the same folder.
- Added publication-oriented README and changelog.
